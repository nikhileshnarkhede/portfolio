📦 Supply Chain Tracker

Full-stack, data-driven supply chain management application with real-time dashboards and role-based access

Project Overview

  • Designed and developed a full-stack, data-driven supply chain management application to track products, vendors, and orders in real time
  • Built interactive dashboards and analytics views using Streamlit and Plotly to visualize operational metrics and trends
  • Implemented role-based access control to support multiple user types with secure login workflows
  • Enabled structured data ingestion and export through CSV upload/download pipelines for seamless data handling
  • Applied data preprocessing and transformation using Pandas and NumPy to support analytics and downstream modeling
  • Demonstrated end-to-end application thinking, integrating data engineering, analytics, and user-facing components
